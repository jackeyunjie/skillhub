# Contradiction Patterns

## Private Domain / Community Commerce

Surface:
- GMV is low
- group has many people but few buyers

False contradiction:
- not enough members

Possible real contradiction:
- weak trust
- no upgrade path
- users are not segmented
- ordinary users do not become active customers

## ToB SaaS / Enterprise Training

Surface:
- renewal is low

False contradiction:
- customer success needs to call more

Possible real contradiction:
- product is not used deeply
- decision makers do not see value proof
- switching cost has not been built

## Offline Clinic / High-Trust Service

Surface:
- many visitors do not buy

False contradiction:
- traffic is not enough or price is too high

Possible real contradiction:
- customer does not perceive urgency
- problem severity is not visible
- treatment plan is not trusted

## Content / Course

Surface:
- conversion or completion is low

False contradiction:
- marketing copy is not strong enough

Possible real contradiction:
- product kernel is unclear
- promise does not match user pain
- content does not create early transformation
