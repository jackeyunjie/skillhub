# Plugin Catalog Reference

## Plugins Mentioned In The Talk

- Hyperframes: video generation
- 飞书接入: messaging and docs workflow
- Computer Use: desktop control
- Chrome: browser session access
- GitHub: repo, issue, PR, CI workflow
- Spreadsheets: spreadsheet read/write and analysis

## Rule

- Install only what unlocks your bottleneck.
- Separate core workflow logic from tool access.
