# Content Factory Reference

## Use This For

- One article to research brief
- One source to whiteboard concepts
- One source to final script

## Pattern

1. Feed a source item into Codex.
2. Let the skill research and analyze.
3. Generate multiple visual directions.
4. Pick one direction.
5. Generate the narration script.
6. Record the video.
7. Edit with a separate skill if needed.
