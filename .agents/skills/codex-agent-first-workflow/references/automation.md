# Automation Reference

## Use This For

- Daily or weekly recurring tasks
- Scheduled intelligence collection
- Small repeatable summaries

## Pattern

1. Describe the task in natural language.
2. Let Codex create the automation.
3. Confirm the generated automation.
4. Manage it from the automation sidebar.
5. Start small, then add complexity.

## Example

```text
每天下午5点，总结今天项目目录里的变更，并输出一份简短Markdown报告。
```
