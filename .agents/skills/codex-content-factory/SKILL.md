# Codex Content Factory

Turn one source item into a production-ready content package.

## Do This

1. Confirm the source item and target audience.
2. Extract the real angle before drafting copy.
3. Separate research from final script generation.
4. Produce multiple visual directions before locking one.
5. Generate a script that matches the chosen visual structure.
6. Hand off editing to a separate workflow if video is being recorded.

## Operating Rules

- Prefer source-backed research over vibes.
- Keep the angle explicit: what is new, why it matters, who should care.
- Use multiple candidate visual directions, not one premature answer.
- When writing-stage quality matters, note whether another model should be used for that phase.
