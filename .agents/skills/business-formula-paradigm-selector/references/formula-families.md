# Formula Families

## Single Transaction

Use when the business closes a meaningful transaction in one decision cycle.

Examples:
- clinic first treatment
- car sales
- home renovation service
- one-off consulting package

Watch:
- problem awareness
- trust
- proposal acceptance
- price acceptance
- decision urgency

## Continuous Repurchase

Use when customer lifetime value depends on repeated behavior.

Examples:
- private domain e-commerce
- membership products
- consumables
- community services

Watch:
- first purchase
- satisfaction
- repeat trigger
- upgrade path
- active customer segments

## Renewal / Subscription

Use when customer success drives renewal.

Examples:
- SaaS
- enterprise training subscription
- annual service contract

Watch:
- activation
- usage depth
- stakeholder adoption
- value proof
- switching cost

## Node Conversion

Use when the bottleneck is a process chain.

Examples:
- landing page to payment
- lead to sales call
- onboarding funnel

Watch:
- node exposure
- node completion
- drop-off
- intent quality

## Continuous Action

Use when repeated behavior matters more than one conversion.

Examples:
- habit products
- workflow tools
- learning programs

Watch:
- trigger
- action completion
- feedback
- repeat rate
