# Codex Knowledge Base Capture

Capture a successful run as reusable knowledge.

## Do This

1. Capture the final task description.
2. Separate raw evidence from conclusions.
3. Extract the stable prompt or contract.
4. Split durable knowledge into:
   - workflow
   - agent boundary
   - skill rule
   - sample case
5. Link the assets into the vault index.

## Operating Rules

- Save the process, not only the final answer.
- Prefer one concrete sample over many vague summaries.
- Split concept, workflow, skill, and agent notes cleanly.
- If the content should be callable by another agent later, bias toward skill packaging.

- Read [references/split-rules.md](references/split-rules.md) to decide what becomes a concept, workflow, skill, agent, or sample.
