# Daily Intel Pattern

Use this pattern for recurring briefings:

1. Collect from the defined source list.
2. Remove duplicates and low-signal items.
3. Extract facts, dates, names, and implications.
4. Draft structured Markdown.
5. Optionally render HTML.
6. Save to the archive path.
7. Return a short run summary with paths and missing sources.

Recommended summary fields:

- markdown path
- html path
- main storyline
- top items count
- candidate follow-up topics
- missing or failed sources
