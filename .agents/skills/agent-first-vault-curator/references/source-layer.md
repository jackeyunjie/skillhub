# Source Layer

Always identify the best available source before deriving notes.

For this repository:

- Prefer `原文数据/教学文档---260612 Agent First：用Codex重新设计你的工作流.md` as the working source.
- Treat external landing doc links as upstream references.
- If upstream fetch is blocked by authentication, record the limitation instead of pretending the source was refreshed.

Use `Sources/` notes to capture:

- source path or URL
- quality of the source
- whether the source is current or stale
- how to refresh it later
