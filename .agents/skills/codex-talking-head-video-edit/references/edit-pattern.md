# Edit Pattern

1. Install the editing skill if missing.
2. Take the source video path.
3. Detect filler, pauses, and repetition.
4. Cut low-value sections.
5. Regenerate subtitles.
6. Return the edited video and final subtitle path.

Representative trigger:

```text
帮我剪这个口播视频，要求去掉废话、停顿和重复内容，重新生成字幕，并把最终成片和字幕文件输出到同目录。
```
