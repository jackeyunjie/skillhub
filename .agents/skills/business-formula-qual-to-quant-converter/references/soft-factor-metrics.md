# Soft Factor Metrics

## Trust

Observable behaviors:
- opens proof material
- stays on proof section
- asks deeper questions
- accepts guarantee terms
- pays without repeated reassurance

Candidate metrics:
- trust proof exposure rate
- proof-section dwell time
- objection rate
- payment completion after proof exposure
- customer story click rate

## Urgency

Observable behaviors:
- books now instead of "later"
- asks about next available slot
- responds quickly
- accepts limited-time action

Candidate metrics:
- time from diagnosis to decision
- booking rate within same day
- response latency
- deadline-triggered conversion

## Value Perception

Observable behaviors:
- compares against alternatives
- asks about outcomes, not just price
- chooses higher-value package
- uses core feature repeatedly

Candidate metrics:
- upgrade acceptance rate
- feature repeat usage
- value-report open rate
- alternative comparison win rate

## Customer Success

Observable behaviors:
- reaches activation milestone
- uses product in real workflow
- decision maker sees result
- team adopts process

Candidate metrics:
- activation completion
- key feature usage frequency
- stakeholder attendance
- value proof delivery rate
- renewal intent with evidence
