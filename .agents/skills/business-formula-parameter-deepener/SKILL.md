# Business Formula Parameter Deepener

Use this when a formula is too shallow to guide action.

## Workflow

1. Capture the current L1-L2 formula.
2. Identify which parameter hides the core contradiction.
3. Split by segment, stage, node, channel, user type, or action.
4. Continue until the parameter names observable behavior or a controllable process.
5. Mark what is actionable now.
6. Mark what needs measurement before action.

## Output Shape

```text
Current formula:
Shallow parameter:
Why it is too shallow:
L3 segmentation / node parameters:
L4 behavior / process parameters:
Priority parameter:
Actionable levers:
Unknowns requiring measurement:
Next action:
```

## Depth Test

The decomposition is not deep enough if the output still says:
- improve conversion
- optimize experience
- increase trust
- improve retention
- strengthen customer success
- increase traffic

Push until it says what behavior, node, trigger, or proof must change.

## Rules

- Stop only when the next action becomes clearer.
- Prefer behavior parameters over abstract business nouns.
- Split averages into meaningful segments before optimizing.
- Mark whether each parameter is input, process, output, or lagging result.
