# Parameter Ladder

## L1 Result Metrics

Examples:
- revenue
- GMV
- profit
- renewal rate
- conversion rate

Use:
- see outcome

Limit:
- cannot guide action.

## L2 Business Subject Metrics

Examples:
- traffic
- conversion
- average order value
- repurchase
- customer count

Use:
- locate rough area

Limit:
- still too broad.

## L3 Segmentation / Stage Metrics

Examples:
- new vs old users
- active vs dormant users
- channel A vs channel B
- first order vs repeat order
- target segment vs non-target segment

Use:
- locate where the problem happens.

## L4 Behavior / Node Metrics

Examples:
- page view completion
- trust proof exposure
- add-to-cart
- payment success
- 7-day activation
- key feature usage
- decision-maker attendance

Use:
- guide concrete action.

## L5 Hypothesis Metrics

Examples:
- which trust proof improves paid conversion
- which trigger drives second purchase
- which onboarding action predicts renewal

Use:
- design experiments.

## L6 Source Insight

Examples:
- true alpha / advantage
- true urgency source
- real switching cost
- hidden demand driver

Use:
- reshape strategy or model.
