# Modeling 85 Essence

Use this skill to build an 85-point essence model: compress a complex topic into a definition that can actually guide decisions.

## Start Here

1. Name the topic.
2. Gather competing explanations, definitions, or schools of thought.
3. Ask what real decision this essence must guide.
4. Compare definitions against real cases.
5. Compress the topic into the shortest useful statement.
6. Test whether that statement can guide at least three real decisions.

## Operating Rules

- Build for future action, not retrospective storytelling.
- Keep compressing until the definition becomes sharp.
- Preserve only elements that change decisions.
- Distinguish the topic from nearby but different concepts.
- Reject elegant phrases that cannot guide real-world choice.

## Compression Ladder

Return all of the following when possible:

1. Topic
2. Competing definitions
3. Your compressed versions:
   - 2-word version
   - 4-word version
   - full working definition
4. Required elements of the essence
5. Three real decisions the essence can guide
6. Boundary: what is often confused with this topic but should be excluded

## Truth Test

An essence is only good if it can explain and guide at least three real decisions.

If it only sounds profound or only explains the past, it is not done.
