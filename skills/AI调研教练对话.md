# 技能：AI 调研教练对话

## 触发条件
- 需要对一个模糊业务问题进行结构化调研
- 想判断某个产品/课程/市场是否值得进入
- 希望把 AI 从「答案生成器」变成「调研协作者」

## 输入
- 调研问题描述（越具体越好）
- 已有事实和资料
- 约束条件（用户、价格、地区、时间）
- 决策目标（这次调研要服务什么决策）

## 处理步骤
1. **澄清问题**：与 AI 一起把模糊问题拆成可研究的调研课题。
2. **生成假设地图**：列出关键假设，区分事实与推理。
3. **第一轮证据补强**：追问 AI 是否有数据和资料支撑。
4. **第二轮时效校准**：追问是否有更近的数据。
5. **第三轮场景校准**：追问是否有更贴近中国/目标用户的数据。
6. **生成决策报告**：输出结论、依据、风险、下一步验证计划。
7. **复盘纠偏**：回顾 AI 的回答过程，记录误判点和改进点。

## 输出格式
```markdown
# 调研报告：XXX

## 1. 调研问题
## 2. 约束条件
## 3. 关键假设（事实 vs 推理）
## 4. 证据清单（含来源、时效、适用边界）
## 5. 核心结论
## 6. 风险与反例
## 7. 下一步验证计划
```

## 质量标准
- [ ] 问题描述清晰，目标决策明确
- [ ] 每个关键结论都有证据或来源支撑
- [ ] 证据经过时效、地域、适用边界校准
- [ ] 区分了事实、推理和假设
- [ ] 报告末尾有明确的下一步验证动作
