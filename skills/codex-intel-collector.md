# Codex Intel Collector Skill

## Purpose

Collect and normalize information from multiple sources for a daily or recurring briefing.

## Based On The Talk

- Codex can run scheduled automation
- The output can be saved as Markdown and then rendered to HTML
- Source coverage should be visible

## Input Contract

- Topic or watchlist
- Source list
- Time window
- Priority rules
- Exclusion rules
- Output path

## Output Contract

- Markdown digest
- Source coverage notes
- Missing-source notes
- Summary of key events

## Operating Rules

- Prefer primary sources
- Mark missing sources explicitly
- Keep dates exact
- Keep the output short enough to scan quickly

## Good For

- Daily intelligence
- Trend monitoring
- Content research prep
