# HTML Brief Generator Skill

## Purpose

Convert a Markdown digest into a visually strong HTML brief.

## Based On The Talk

- The daily briefing should become a polished HTML report
- High information density matters
- The page should be responsive and easy to scan

## Input Contract

- Title
- Date
- Markdown digest
- Styling direction

## Output Contract

- A single HTML page
- Clear information hierarchy
- Cards or section blocks
- Source state visibility

## Rendering Priorities

- Dense but readable
- Editorial feel
- Light/dark adaptability if needed
- Top events visible quickly
