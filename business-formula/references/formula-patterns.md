# 十大公式范式（2.0 版）

> 来源：一堂业务公式课程 - Truman 主讲
> 以提升为追求，三类提升类型：收入提升类、竞争提升类、运营提升类

```
        ┌─────────────────┐
        │   收入提升类      │ ← 顶层：商业模式/财务模型
        └────────┬────────┘
                 │
    ┌────────────┼────────────┐
    │            │            │
┌───┴───┐   ┌───┴───┐   ┌───┴───┐
│竞争提升类│   │运营提升类│   │（支撑层）│
└───────┘   └───────┘   └───────┘
```

> 建议：长期盯顶层收入和更本质的因果关系；定期关注其他角度。能拆哪个拆哪个，哪个认知多重点盯哪个。

## 第一类：收入提升类

### 1. 流量广告变现
```
流量收入 = 广告流量 × 转化率 × 单次点击价格
```

### 2. APP 推广变现
```
APP 收入 = 各市场用户数 × 看到率 × 下载率 × 留存率 × 付费率
```

### 3. 电商获客变现
```
GMV = 曝光量 × 点击率 × 加购率 × 下单率 × 支付成功率 × 无退货率
```

### 4. 企业咨询转化
```
收入 = 目标客户数 × 转化率 × 咨询费 × 客均购买次数
```

### 5. SaaS 转化
```
成单量 = 线索量 × 电话接通率 × 邀约上门率 × 订金率 × 定金转化率
```

### 6. 加盟商转化
```
成功合作数量 = 文章流量 × 咨询率 × 面谈率 × 签约合作率
```

### 7. 线下零售
```
收入 = 客流量 × 进店率 × 转化率 × 客单价 × (1-退货率)
```

### 8. 用户生命周期（AARRR/RARRA）
```
获客 → 激活 → 留存 → 收益 → 传播
```

## 第二类：竞争提升类

### 9. 黏性 / 脱离成本类

**在线教育：**
```
脱离成本 = 课程 + 作业 + 活动 + 社交 + 方法论
```

**社交产品：**
```
脱离成本 = 社交关系链 + 个性化信息 + 定制化配置
```

**SaaS 产品：**
```
脱离成本 = 业务数据 + 使用习惯 + 定制化服务
```

### 10. 规模化生产类

**电商选品：**
```
爆款数 = 选品库 × 评估通过率 × 测试点击率 × 粉丝投票率 × 预售通过率 × 选出爆款率
```

**短视频创作：**
```
选题数 = 选题池容量 × 脚本通过率 × 拍摄效率 × 算法适配率
```

## 第三类：运营提升类

### 11. 留存节点类

**付费订阅：**
```
留存率 = 1日留存率 × 7日留存率 × 30日留存率
```

**年费会员：**
```
续费率 = 1年续费率 × 2年续费率 × 3年续费率
```

### 12. 续费 / 复购类

**减重私教：**
```
续费率 = 14日体测达标率 × 30日打卡率 × 90日体重下降率
```

**职业教育：**
```
完课率 = 7日打开率 × 21日章节完成率 × 60日考证通过率
```

### 13. 转化漏斗类

**直播带货：**
```
转化率 = 停留达标率 × 商品点击率 × 加购率 × 支付率
```

**在线阅读：**
```
付费率 = 章节打开数 × 免费章读完率 × 付费弹窗点击率 × 支付成功率
```

### 14. 高客单价长链路类

**在线问诊：**
```
成单率 = 首诊转化率 × 复诊率 × 处方率
```

**留学中介：**
```
成功率 = 背景评估率 × 选校确认率 × 文书定稿率 × Offer获取率 × 签证通过率
```

## 切分维度参考

当用户不确定如何切分时，提供以下 5 种常见切分维度：

| 维度 | 示例 |
|------|------|
| **按渠道** | A渠道GMV + B渠道GMV + C渠道GMV |
| **按用户类型** | 新用户GMV + 复购用户GMV + 转介绍GMV |
| **按生命周期** | 获客GMV + 激活GMV + 留存GMV + 收益GMV |
| **按场景** | 工作日午餐GMV + 家庭聚餐GMV + 外卖GMV |
| **按 SKU** | 爆款GMV + 常规款GMV + 新品GMV |
